# Spring Cloud

本章节的类容我们主要讨论了关于 Spring Cloud 微服务开发的项目组件，其中包括的组件如下

**注册中心**

* Spring Cloud Eureka
* Spring Cloud Zookeeper
* Spring Cloud Consul
* Spring Cloud Ribbon

**服务调用**

* Spring Cloud OpenFeign

**服务降级、服务限流**

* Spring Cloud Hystrix

**服务网关**

* Spring Cloud Gateway

**配置中心**

* Spring Cloud Config

**服务总线**

* Spring Cloud Bus

**链路追踪**

* Spring Cloud Stream
